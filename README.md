# 🚀 Spring Boot Journey

A complete learning repository for mastering **Spring Boot, Spring Framework, and Hibernate** based on the course:

> *Spring Boot 4, Spring 7 & Hibernate for Beginners - Chad Darby*

---

## 📚 About This Repository

This repository contains my personal learning journey, including:

* 📖 Course explanations
* 🧠 Personal notes & summaries
* 🔗 Useful resources
* ✅ Progress tracking for each section

The main goal is to deeply understand how to build real-world backend applications using Spring Boot.

---

## 🎯 Course Objectives

* Build Spring Boot applications from scratch
* Understand Dependency Injection & IoC
* Work with Hibernate & JPA
* Build REST APIs
* Create MVC applications
* Apply Spring Security
* Handle real-world database operations

> Based on the course content
